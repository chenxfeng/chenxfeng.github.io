关于VisuAlgo

Posted on 2017-08-30

视觉算法(VisuAlgo)是一个将数据结构与算法通过动画展现出来的网站。

风格

内容

  1. 排序 Sorting

  2. 位图 Bitmask

  3. 链表 Linked List

  4. 哈希表 Hash Table

  5. 二叉堆 Binary Heap

  6. 二叉搜索树 Binary Search Tree

  7. 图 Graph Structures

  8. 并查集 Union-Find DS

  9. 线段树 Segment Tree

  10. Fenwick Tree

  11. 递归树/有向无环图 Recursion Tree/DAG

  12. 图的遍历 Graph Traversal

  13. 最小生成树 Min Spanning Tree

  14. 单源最短路径 SS Shortest Paths

  15. 网络流 Network Flow

  16. 图的匹配(对集) Graph Matching

  17. Cycle Finding

  18. 前缀树 Suffix Tree

  19. 前缀数组 Suffix Array

  20. 几何(多边形) Geometry (Polygon)

  21. 凸包 Convex Hull

  22. 最小顶点覆盖 Min Vertex Cover

  23. 旅行商问题 Traveling Salesman

其它

多种语言支持(中文似乎还不太完善)

一部分模块有训练题目

You can visualize the recursion tree (or DAG, if there are overlapping subproblems and Dynamic Programming (DP) is applicable) of ANY valid recursive function that can be written in JavaScript.
