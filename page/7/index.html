伯努利随机试验:

  • $Pr{X \geq m} = \sum{j=m}^n (_j^n)p^jq^{n-j}$

切尔诺夫界/不等式(Chernoff bounds):

  • $P_r{ X \leq (1 - \epsilon)pn } \leq e^{-\epsilon^2np/2}$
  • $P_r{ X \geq (1 + \epsilon)pn } \leq e^{-\epsilon^2np/3}$
